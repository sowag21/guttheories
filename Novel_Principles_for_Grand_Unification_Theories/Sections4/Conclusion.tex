We studied chiral gauge-Yukawa theories' ultraviolet behaviour, specifically the generalised Georgi--Glashow (GG) and the Bars--Yankielowicz (BY). The focus was on completely asymptotically free (CAF) regimes for the models with either a scalar in the anti-fundamental or the adjoint representation. Additionally, we wanted to extend the model to include multiple chiral families to see how this changed the CAF regions.

Our work was based on two approaches to CAF analysis and extended the general CAF analysis for multiple couplings \cite{Giudice:2014tma} to consider the Yukawa coupling both on and off fixed flow. This proved crucial for the CAF regions of the BY (GG) model with one adjoint scalar, as it only exhibits CAF for the Yukawa coupling of fixed flow for $N_g\in\{1,2\}$ ($N_g\in\{1,\dots,4\}$) chiral families. All models exhibited CAF for some $(N,p)$. A summary of the CAF regions for both models GG and BY can be found in Tables \ref{tab:conc_part1} and \ref{tab:conc_part2}.

We derived the functions $p_{\max}$ and $p_{\min}$ that characterise the CAF regions across all the considered models. This formulation makes the analysis applicable to model building, allowing one to insert the number of colours and chiral families and then obtain the required amount of vector-like fermions for a theory to be CAF. 

This analysis elucidates a framework for choosing chiral gauge-Yukawa theories which are UV-complete by restricting the number of vector-like fermions. This framework has broad applications within model-building and beyond the Standard Model physics. \comment{Next steps?}
